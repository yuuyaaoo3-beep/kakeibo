Suica 履歴取得（Claude in Chrome） 半自動
① Claude に指示をコピー
② Suica で実行
③ JSON を貼り付け

① Claude in Chrome に以下の指示を貼り付けて実行してください

モバイル Suica のマイページ（https://www.mobilesuica.com/）を開いてください。
すでにログイン済みの場合はそのまま進めてください。

「ゆう」カードの利用履歴ページ（以下のURLを直接開いてください）に移動してください：
https://www.mobilesuica.com/iq/ir/SuicaDisp.aspx?returnId=SFRCMMEPC03

表示されている利用履歴テーブルの全データを読み取り、以下の JSON 形式で出力してください。

[
  {
    "date": "YYYY-MM-DD",
    "memo": "利用場所（入口 → 出口）または内容",
    "amount": 金額（数値・マイナスなし・円単位）,
    "isStation": true,
    "isCharge": false
  }
]

ルール：
- 月日は今年（2026年）として YYYY-MM-DD 形式に変換
- 鉄道・地下鉄・バスなど交通機関は isStation: true
- 物販・コンビニなどは isStation: false
- チャージ（「チャージ（モバイル）」など）は isCharge: true、isStation: false
- 金額は数値のみ（カンマ・円記号・マイナス符号なし）
- 金額が 0 の行も含めて出力する
- 「繰越」行は除外する
- JSON のみ出力し、説明文は不要

② 実行時の注意

  • Claude in Chrome を開いた状態で上記を貼り付けて送信
  • すでに mobilesuica.com にログイン済みであればそのまま使える
  • 1ページに最大 100 件表示されます
  • チャージ（+金額）はインポート時に自動スキップされます
  • 「物販」はカテゴリ「買い物・日用品」として取り込まれます（後で変更可）
